# del2dup

指定したフォルダ内で同一内容の重複ファイルを検出し、最も深い階層にある1つだけを残す CLI ツール。

## インストール

```sh
cargo build --release
```

## 使い方

```
del2dup -e <拡張子> <フォルダ1> <フォルダ2> ... [--delete]
```

- `<拡張子>` : 対象とするファイルの拡張子（ドットなし、`-e` で複数指定可）
- `<フォルダ>` : 検索対象のフォルダ（複数指定可）
- `--delete` : 指定すると実際に重複ファイルを削除する。省略時はドライラン。

### 例

```sh
# ドライラン: a と b の拡張子を対象に AAAA, BBBB フォルダを検索
del2dup -e a -e b AAAA BBBB

# 実際に削除
del2dup --ext txt --delete /path/to/dir1 /path/to/dir2
```

## 重複時の選択ルール

1. 入力フォルダからの相対的な**ディレクトリ階層が最も深い**ファイルを残す
2. 深さが同じ場合は、引数で**先に指定したフォルダ**にあるファイルを優先

例: `AAAA BBBB` の順で指定し、`AAAA/NNNN/KKKK/12.a` と `BBBB/MMMM/11.a` が同一内容の場合、AAAA 側（深さ3）が BBBB 側（深さ2）より深いため AAAA 側を残す。
